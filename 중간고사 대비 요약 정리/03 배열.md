# 배열

## 리스트

: 원소들을 한 줄로 나열한 구조 (특별한 순서를 따름)

- 유한한 원소들의 나열
- 각 원소들은 인덱스에 대응됨* (가장 중요한 성질)

## 리스트의 구현 방법

- **배열**(Array): **인덱스**에 기반한 구현
    - 메모리에 배열의 크기보다 더 큰 **공간이 허용될 때** 사용
- **연결 리스트**(Linked List): **포인터**에 기반한 구현
    - 메모리에 배열의 크기보다 더 큰 **공간이 없을 때** 사용

<sub>변수: 주소에 1:1 대응되는 니모닉</sub>

## 배열의 기본 연산

(프로그래밍 언어에서 제공)
- 생성 (create)
- 인출 (retrieve)
- 저장 (store)
